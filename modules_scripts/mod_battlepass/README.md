# Battle Pass


## Description

This is a simple ingame battle pass system i made for oregoncore, this will award players on tier up for completing quests and pvp


## How to use ingame


## Requirements

My new module requires:

- OregonCore Restructured


## Installation

```
1) Simply place the module under the `modules` directory of your OregonCore source. 
2) Import the SQL manually to the right Database (auth, world or characters) 
3) Re-run cmake and launch a clean build.
```

After inputting all SQL files, fill out the battlepass_reward table in your world database for each tear you require

you will see an option column on the table

option 1 would reward the player an item
option 2 would reward the player gold/money
option 3 will level the player
